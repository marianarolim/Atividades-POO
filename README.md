# 📚 Exercícios de JavaScript - POO

Repositório com exercícios práticos de **Programação Orientada a Objetos (POO)** em JavaScript.

---

## 🚀 Conceitos praticados

* Classes e Objetos
* Métodos e `constructor`
* Herança (`extends`)
* Polimorfismo
* Encapsulamento (`#privado`)
* Arrays + loops (`for`, `forEach`)

---

## 📌 Exercícios

* 💰 Conta Bancária (depósito, saque, saldo)
* 🐶🐱 Animais (polimorfismo)
* 🎓 Alunos (cálculo de média)
* 📚 Livros (array de objetos)
* 🔐 Usuário (senha privada)
* 🚗 Veículos (herança)
* 🛒 Carrinho (adicionar e calcular total)

---

## 🛠 Tecnologias

* JavaScript (ES6+)
